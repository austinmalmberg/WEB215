What's the difference between <i> and <em>?

While both tags are typically (though not always) displayed as italic, <i> is intended to separate while <em> is meant to distinguish.
The same principle applies to <b> and <strong> tags.

Examples:

Using <i>: Mama says "Stupid is as stupid does."

Using <em>: Run, Forrest! Run!

With accessibility in mind, front-end developers should consider using <strong> and <em> in place of <b> and <i>.
